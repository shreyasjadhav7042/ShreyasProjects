# Freelancer Performance Dashboard

## Project Overview

The Freelancer Performance Dashboard is a data analytics project designed to analyze and visualize key performance metrics of freelancers across multiple skill domains. The objective of this project is to identify trends in freelancer earnings, experience levels, ratings, client satisfaction, and workforce distribution while demonstrating an end-to-end data analytics workflow.

The project begins with raw freelancer data stored in CSV format. The dataset was inspected and validated using Microsoft Excel, followed by data cleaning and preprocessing in Python using Jupyter Notebook. After preparing the dataset, SQL was used to perform analytical queries and generate insights related to freelancer performance, skill distribution, hourly rates, and client satisfaction. Finally, Power BI was used to develop an interactive dashboard that transforms the analyzed data into meaningful visualizations and business insights.

The dashboard enables users to explore freelancer demographics, compare hourly rates across skills, evaluate client satisfaction levels, analyze active freelancer participation, and understand the relationship between experience and earnings. Through KPI cards, interactive filters, and multiple visualizations, the dashboard provides a comprehensive overview of freelancer market trends.

This project demonstrates practical skills in data cleaning, data transformation, SQL analysis, data visualization, dashboard design, and business intelligence reporting using industry-standard tools such as Excel, Python, SQL, and Power BI.

## Project Workflow

### 1. Raw Data Collection

The project begins with a raw freelancer dataset stored in CSV format and reviewed using Microsoft Excel.

### 2. Data Cleaning & Preprocessing

Using Python and Jupyter Notebook, the dataset was cleaned and prepared for analysis. This included handling missing values, data validation, formatting corrections, and ensuring consistency across records.

### 3. Data Analysis

SQL queries were used to analyze freelancer performance and answer business-related questions regarding skills, hourly rates, ratings, experience levels, and client satisfaction.

### 4. Dashboard Development

Power BI was used to create an interactive dashboard that presents key metrics and visual insights through charts, KPI cards, filters, and analytical visualizations.

## Key Insights

* DevOps freelancers have the highest average hourly rates.
* Client satisfaction scores remain consistently high across most skills.
* Hourly rates are relatively stable among different skill categories.
* Gender-based differences in average hourly rates are minimal.
* More experienced freelancers generally command higher hourly rates.
