# 🩺 Diabetes Prediction System

## 📌 Project Overview
This project predicts whether a patient is likely to have diabetes using Machine Learning. Multiple algorithms were trained and compared, and the Decision Tree model achieved the best performance on the test dataset.

---

## 🚀 Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Hyperparameter Tuning using GridSearchCV
- Comparison of Multiple ML Models
- Streamlit Web Application
- Diabetes Prediction

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- Streamlit
- Joblib

---

## 🤖 Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest
- XGBoost

---

## 🏆 Best Model
**Decision Tree**

- Training Accuracy: **81%**
- Testing Accuracy: **76.62%**

---

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run application.py
```

---

## 📂 Dataset

Diabetes dataset containing the following features:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

Target:
- 0 = Not Diabetic
- 1 = Diabetic
